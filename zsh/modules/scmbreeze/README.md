# scmbreeze

scmbreeze (SCM Breeze) integration for zsh dotfiles.

## Requirements

- zsh

## Installation

This module is included in the dotfiles and loaded automatically.

## Usage

SCM Breeze provides enhanced git integration with numbered file shortcuts and improved `git status` output.

## Environment

- `ZSH_SCMBREEZE_PATH` — Path to the scmbreeze module
