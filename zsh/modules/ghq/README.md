# ghq

[ghq](https://github.com/x-motemen/ghq) integration for zsh dotfiles.

## Requirements

- [ghq](https://github.com/x-motemen/ghq) - Remote repository management
- [peco](https://github.com/peco/peco) or [fzf](https://github.com/junegunn/fzf) (optional, for interactive selection)

## Installation

This module is included in the dotfiles and loaded automatically.

## Usage

Provides ghq integration with interactive repository lookup.

## Environment

- `ZSH_GHQ_PATH` — Path to the ghq module
- `GHQ_ROOT` — Root directory for ghq-managed repositories
