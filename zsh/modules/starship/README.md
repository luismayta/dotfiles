# starship

Starship prompt integration for zsh dotfiles.

## Requirements

- [starship](https://starship.rs/) - The minimal, blazing-fast, and customizable prompt

## Installation

This module is included in the dotfiles and loaded automatically.

## Usage

Starship provides a customizable prompt with various modules.

## Environment

- `ZSH_STARSHIP_PATH` — Path to the starship module
- `STARSHIP_CONFIG` — Path to the starship configuration file
